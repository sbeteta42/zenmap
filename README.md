# zenmap
script d'installation automatisée de zenmap pour Debian/Ubuntu/Kali

# Pré-requis
OS: Debian/Ubuntu/Kali avec Interface GUI
```bash
apt install git- y
```
# Installation
```bash
git clone https://github.com/sbeteta42/zenmap/main/zenmap.git
chmod +x install_zenmap.sh
./install_zenmap.sh
```
